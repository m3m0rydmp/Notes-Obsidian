## Introduction to Web Security
Attacks are extremely frequent where it:
* Target Applications
* Target applications' users
Attacks are not always publicised, and some companies are taking it seriously.

A secure Hyper Text Transfer Protocol is important (HTTPS) or Secure SSL. More feature means more attack surface such as:
* Parameters
* Verbs
* URI
* Headers
* Cookies
* Uploads
* APIs
* Mobile Applications
## Mapping the Attack Surface
* **Web Spidering**
* **Hidden Content**
* Content Discovery through Google Dork or Web Archives
* Response Headers
* Stack: Error Messages
* Understand Functionality: Dissect Request
